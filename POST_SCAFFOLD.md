# Welcome to your new web app!

Your project is ready inside the S/ASH workspace.

## Quick start

```bash
pnpm dev
```

This starts the dev server on port **5173**. The preview panel in your workspace will show the app once it's ready.

## Where to edit code

- **Routes** live in `app/routes/` (e.g. `app/routes/_index.tsx`)
- **Database schema** is in `app/db/schema.ts`
- **UI components** are in `app/components/ui/` (shadcn/ui)
- **Auth config** is in `app/lib/auth.server.ts`

## Deploy

When you're ready, click the **Deploy** button in the workspace header, pick a slug, and hit **Deploy**. Your live URL will be something like `https://my-app.app.slash.ai.kr`.

See `README.md` in this project for full docs on environment variables, database commands, and production setup.
