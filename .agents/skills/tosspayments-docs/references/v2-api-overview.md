## 카테고리 가중치 (category.ts)

BM25 점수에 곱해지는 카테고리별 가중치입니다.

```
guides: 1.2   // 가이드 문서 우선순위 높임
reference: 1.0 // 레퍼런스 문서 기본값
sdk: 1.0       // SDK 문서 기본값
resources: 0.8 // 리소스 문서 약간 낮춤
blog: 0.7      // 블로그 포스트 낮춤
codes: 0.5     // 에러코드/벤더사/enum 정보 많이 낙임
legacy: 0.4    // 레거시 문서 많이 낙임
```

## 검색 모드 BM25 파라미터 (search-mode.ts)

```
broad: { k1: 1.0, b: 0.5 }   // 낮은 k1 = 용어 빈도에 덜 민감, 낮은 b = 문서 길이에 덜 민감
balanced: { k1: 1.2, b: 0.75 } // 표준 BM25 값
precise: { k1: 1.5, b: 0.9 }  // 높은 k1 = 용어 빈도에 더 민감, 높은 b = 문서 길이에 더 민감
```
